# Profile Card — Bug Fix Task

Entry-level Web Track task for Deimos Tech internship. Fixed 8 UI bugs in a starter profile card component so it matches the provided design spec.

## Bugs Fixed
1. Centered the card on the page using flexbox
2. Fixed avatar border-radius so it renders as a proper circle
3. Centered name and role text inside the card
4. Added the missing divider line between role and button
5. Corrected button background color to match spec (#533483)
6. Added a hover state for the button with a darker shade
7. Adjusted spacing to follow a consistent 4px rhythm
8. Increased card border-radius to match spec (16px)

## What Changed and Why
Updated CSS values and HTML structure in `starter_profile_card.html` so the component visually matches `design_spec_profile_card.png`. Also fixed an unclosed CSS comment that was silently breaking styles below it, and added a viewport meta tag for mobile responsiveness.
